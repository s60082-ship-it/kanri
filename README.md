# お金管理 - Kanri

シンプルで使いやすいお金管理アプリです。支出・収入の記録、予算管理、月ごとの統計が簡単にできます。

## ✨ 特徴

### 📝 記録機能
- 支出・収入の記録
- カテゴリ分類（食事、交通、エンタメ、医療、その他）
- メモ付きで詳細を記録
- 日付を指定できる

### 📊 予算管理
- カテゴリごとに月予算を設定
- 使用状況を視覚化（進捗バー）
- 予算超過時の警告表示
- 月ごとの自動リセット

### 📈 統計・分析
- 月間の収入・支出・残高を表示
- カテゴリ別の支出内訳
- 支出比率（％）を表示

### 📱 PWA対応（iPhone対応）
- ホーム画面にアプリとして追加可能
- オフラインでも使用可能
- ネイティブアプリのような体験

### ☁️ Google Sheets 連携
- データをバックアップ可能
- Google Sheetsへのエクスポート対応

### 🔐 プライバイト保護
- すべてのデータはローカルに保存
- 外部サーバーへの自動送信なし

## 🚀 使い方

### ブラウザでの使用
1. アプリを開く
2. 各タブから機能を選択

### iPhone でアプリ化
1. Safari でこのアプリを開く
2. 「共有」ボタンをタップ
3. 「ホーム画面に追加」をタップ
4. ホーム画面のアイコンをタップで起動

### Android でアプリ化
1. Chrome でこのアプリを開く
2. メニュー → 「アプリをインストール」
3. ホーム画面のアイコンをタップで起動

## 📖 機能説明

### 📝 記録タブ
- **種類** - 支出または収入を選択
- **カテゴリ** - 費目を選択
- **金額** - 金額を入力
- **日付** - 日付を選択（デフォルトは本日）
- **メモ** - オプションで備考を追加

### 📊 予算タブ
- **カテゴリ** - 予算を設定する費目を選択
- **月予算** - 月間の予算額を入力
- 予算と実績を比較
- 進捗バーで視覚化

### 📈 統計タブ
- 月間の収入合計
- 月間の支出合計
- 収支残高
- カテゴリ別支出内訳

### ⚙️ 設定タブ
- **Google Sheets連携** - データをバックアップ
- **月のリセット** - 今月のデータをリセット
- **データ管理** - JSONファイルとしてエクスポート

## 💾 データ保存
- **ローカルストレージ** - IndexedDB を使用して安全に保存
- **バックアップ** - JSON形式でエクスポート可能
- **Google Sheets** - オプションで連携可能

## 🔐 プライバシー
- すべてのデータはブラウザのIndexedDBに保存されます
- サーバーに自動的に送信されません
- Google Sheets連携は手動で実行される場合のみ

## 🌐 対応ブラウザ
- Chrome / Edge（Windows, Mac, Android）
- Safari（iOS 14.5以上）
- Firefox（Windows, Mac, Android）
